# marimo-edit

Open a marimo notebook in the interactive editor.

Usage: /marimo-edit <notebook.py>

```bash
uvx marimo edit $ARGUMENTS
```

This command opens a marimo notebook in the interactive browser-based editor
where you can modify cells and see reactive updates.

## Examples

- `/marimo-edit notebook.py` - Open notebook in editor
- `/marimo-edit --headless notebook.py` - Run editor without opening browser
